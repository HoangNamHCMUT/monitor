TEMPERATURE & HUMIDITY SUPERVISOR

This website is provided to user for supervising the temperature and humidity changes

    Some instructions to use this Website:

  • * Make sure you have Register/Login your account before using this website.
  • * Click the Expand button to open the side bar buttons.
  • * Enter the IP address of the ESP32 Webserver to use the Realtime communication mode.
  • * In the Realtime communication mode, you can transfer data directly between website and ESP32 (such as sending command, getting status, sending backend IP address ...).
  • * For filtering purpose, clicking Open Filter button to open the filter table. Then choosing which kind of data you wanna filter (tempearture or humidity) by clicking on the button next to the Open Filter button. After that, choosing the appropritate filter option (based on time/value).
  • * Double click on Logout button for logout.
